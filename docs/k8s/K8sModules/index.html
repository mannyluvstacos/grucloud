Kubernetes Modules

A Module is a collection of resources packaged with npm, the node package manager. These modules can be imported and be used in your project:

  • @grucloud/module-k8s-cert-manager: provides a set of kubernetes resources to manage SSL certificates, a dependency of the Amazon Load Balancer Controller.
  • @grucloud/module-k8s-load-balancer: Automates the Amazon Load Balancer Controller
  • @grucloud/module-k8s-postgres: Defines resources to setup a Postgres database.
  • @grucloud/module-k8s-redis: Defines resources to setup Redis.
  • @grucloud/module-k8s-web-ui-dashboard: Setup the Kubernetes Web Ui Dashboard
  • @grucloud/module-k8s-prometheus: Monitoring with Prometheus
  • @grucloud/module-k8s-crunchy-postgres: Manages the Crunchy Data Postgres Operator

See the k8s examples how on to consume these modules in your project.
